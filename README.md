Este teste foi desenvolvido em React Native e pode ser executado no GitPod, adicionando o prefixo `gitpod.io/#/` no início deste URL ou no Visual Studio Code, clonando este repositório e seguindo todas as instruções deste pequeno manual.

## execução

!!! Atenção !!! : Antes de rodar este aplicação na IDE online do GitPod ou no VS Code, instalar as dependências para não causar erros na execução da aplicação web. Comandos a serem inseridos no terminal:

    npm install

    npm install bootstrap

    npm install jquery

    npm start

- Pronto! Se você inseriu todos os comandos acima no terminal do VS Code, você estará pronto para testar a mais moderna aplicação de tarefas desenvolvido em TypeScript/JavaScript por mat3ed18.
